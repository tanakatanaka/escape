luaからC言語の構造体をいじる手順
=================================

1. struct { ... }; の宣言をまるごとcppからhに移して公開する
2. 同じ内容をpkgにコピペする
　（luaにいじられたくない変数は削ってもいい）
3. make_lua_header.bat を実行

これでCの構造体の変数がluaで使えます
読み書き両方OK
selfでなくて変数名があるので
C言語ではself->xみたいに書く代わりに、
luaではroom.x みたいに書きます
